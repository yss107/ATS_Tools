# Classic Resume Template

A professional, clean, and traditional resume template built with HTML and CSS. Perfect for job seekers who want a classic, timeless design that works well both on screen and in print.

## Features

- **Classic Design**: Traditional, professional layout that never goes out of style
- **Print-Optimized**: Formatted for standard 8.5" x 11" paper with proper margins
- **Responsive**: Mobile-friendly design that adapts to different screen sizes
- **Easy to Customize**: Simple HTML structure and well-organized CSS
- **Professional Sections**: Includes all standard resume sections
- **Clean Typography**: Uses Times New Roman for a professional appearance

## How to Use

1. **Clone or Download**: Get this repository to your local machine
2. **Edit Content**: Open `index.html` and replace the sample content with your information:
   - Update the name and contact information in the header
   - Modify the professional summary to reflect your experience
   - Replace work experience entries with your job history
   - Update education section with your degrees and institutions
   - Customize the skills section with your technical abilities
   - Add or remove certifications as needed

3. **Customize Styling** (Optional): Edit `style.css` to adjust:
   - Colors and fonts
   - Spacing and layout
   - Section organization

4. **View Your Resume**: Open `index.html` in any web browser to see your resume

## Sections Included

- **Header**: Name and contact information
- **Professional Summary**: Brief overview of your experience and goals
- **Work Experience**: Job history with company, title, dates, and accomplishments
- **Education**: Degrees, institutions, and relevant academic information
- **Technical Skills**: Programming languages, frameworks, tools, and methodologies
- **Certifications**: Professional certifications and credentials

## Printing Tips

- Use your browser's print function (Ctrl/Cmd + P)
- Select "More settings" and ensure margins are set to "Minimum"
- Choose "Save as PDF" for digital distribution
- The template is optimized for single-page resumes but will flow to multiple pages if needed

## Customization Examples

### Changing Colors
To change the accent color, modify the CSS variables in `style.css`:
```css
.section-title {
    color: #2c3e50; /* Change this to your preferred color */
}
```

### Adding New Sections
To add a new section (e.g., "Projects" or "Volunteer Work"), follow this pattern:
```html
<section class="section">
    <h2 class="section-title">Your Section Title</h2>
    <div class="section-content">
        <!-- Your content here -->
    </div>
</section>
```
