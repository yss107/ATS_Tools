# Demo: Customizing Your Classic Resume

This file demonstrates how to customize the classic resume template for different job applications.

## Quick Start Guide

1. **Basic Information Update**
   - Open `index.html` in any text editor
   - Find the header section and update:
     - Name: Change "John Doe" to your name
     - Email: Replace with your email address
     - Phone: Replace with your phone number
     - LinkedIn: Update with your LinkedIn profile
     - Location: Change to your city and state

2. **Professional Summary**
   - Tailor this section for each job application
   - Highlight skills and experience relevant to the specific role
   - Keep it concise (2-3 sentences)

3. **Work Experience**
   - List jobs in reverse chronological order (most recent first)
   - Focus on achievements with quantifiable results
   - Use action verbs to start each bullet point
   - Tailor bullet points to match job requirements

4. **Skills Section**
   - Prioritize skills mentioned in the job posting
   - Group similar skills together
   - Be honest about your skill level

## Example Customizations

### For a Frontend Developer Role
- Emphasize JavaScript frameworks, CSS, and responsive design
- Highlight user experience and interface projects
- Include portfolio or GitHub links

### For a Backend Developer Role
- Focus on server-side languages and databases
- Emphasize API development and system architecture
- Highlight performance optimization achievements

### For a Full-Stack Developer Role
- Balance frontend and backend technologies
- Show experience with complete project lifecycle
- Demonstrate versatility across the tech stack

## Tips for Success

- **Keep it concise**: Aim for 1-2 pages maximum
- **Use keywords**: Include terms from the job posting
- **Quantify achievements**: Use numbers and percentages when possible
- **Proofread carefully**: Check for spelling and grammar errors
- **Update regularly**: Keep your resume current with recent accomplishments

## Print-Ready Format

This template is optimized for printing:
- Standard 8.5" x 11" page size
- Professional margins
- Print-friendly colors
- Readable fonts

To create a PDF:
1. Open `index.html` in your browser
2. Press Ctrl+P (Cmd+P on Mac)
3. Select "Save as PDF"
4. Adjust margins to "Minimum" for best fit
